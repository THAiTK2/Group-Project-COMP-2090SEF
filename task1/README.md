## Introduction 
In contemporary education enviroments, teachers manage many sports team using Excel or paper. This is very slow because the diversification of sports teams has outpaced traditional adminstrative methods. This project aims to improve  the following problems.

1. we want to increasing the administrative efficiency. Teachers need to spend more time to manage scattered Excel sheets and manual registrations. MSUTMS centralizes the data including students, coaches and team through a single interface.

2. we also want to validate eligibility compliance automatically. Since there is big chance of human prone error that using manual verification of GPA and attendance. MSUTMS can block students who failing academic or attendance in real time.

3. Third, there is phenomenon that only teachers who manage sports team can know the team situtation deeply but teachers from other department cannot. It is hard for teachers who response with administration, to have a macroscopic view of performance and win rates across different sports. They cannot distribute the resources easily aim to enhance some teams quality. MSUTMS can show different team standardized report.

4. Finally, there is a risk of data lossing or versioning issues when used local spreadsheet files. MSUTMS use JSON file to store the data and update the data easily.
